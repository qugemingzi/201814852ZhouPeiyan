  总体流程为
  
  {
  
    划分数据集为训练以及测试用；
    
    采用jieba和nltk分词，生成通过停用词，词频长度，字母限制，词频过小等限制的词频词典；

    根据倒排索引词典，词频最大统计，结合tf-idf公式生成文本向量表示；
    
    根据文本向量表示，采用knn分类方法，设置不同的k值，观察方法分类成功率。
    
  }
